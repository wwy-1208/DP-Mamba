# DP-Mamba
The DP-Mamba code for the paper "[A DUAL-PATH MAMBA WITH FIXED AND VARIABLE PATCHES FOR TIME SERIES FORECASTING]
# Environment
* python            3.10.13
* torch             2.1.1+cu118
* mamba-ssm         1.1.1
The installation of mamba-ssm package can refer to https://github.com/state-spaces/mamba.
# Dataset
You can download all the datasets from the "[Autoformer](https://drive.google.com/drive/folders/1ZOYpTUa82_jCcxIdTmyr0LXQfvaM9vIy)" project. Creatae a `data` folder in the current directory and put the downloaded datasets into `data` folder.
